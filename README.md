# node-auth


A node.js authentication system using express nodemailer and mongodb implementing jwt and session management.

## Installation
```
npm install
```

## Usage
```
npm run dev
```
## features
- [x] User registration
- [x] User login
- [x] User logout
- [x] User email verification
- [x] user Authorization
- [x] User Password hashing
- [x] User OTP
